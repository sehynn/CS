## 슬라이딩 윈도우
구간을 한 칸씩 밀면서 탐색하는 기법 

파이썬은 씨처럼 메모리 주소 포인터 안쓰고 그냥 인덱스 변수로 함 

## set()
중복을 허용하지 않는 자료구조 
- 검색 속도
- 리스트 : 최악의 경우 O(n)
- 셋 : 평균 O(1)

## 연결 리스트
- ListNode() : 시작 노드 만들기
- A.val : 노드의 값 가져오기
- A.next : 다음 노드로 이동하기 


## 정렬 : 버블, 선택, 삽입, 퀵, 병합


## 탐색 : 이진 탐색, DFS, BFS

## 그리디, 백트래킹, 동적 계획법
